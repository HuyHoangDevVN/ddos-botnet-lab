# Safety Policy

This repository is simulation-only. Offensive behaviors are prohibited.

## Mandatory Rules

- Keep `SIMULATE_ONLY=true`.
- Run only on localhost or approved lab-only subnets.
- Use only allowlisted commands.
- Keep audit logging enabled.

## Forbidden Activities

- Any real-world offensive execution or adaptation.
- Exposure of services to public networks.
- Bypassing startup guardrails.
